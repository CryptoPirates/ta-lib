# Technical Analysis Library (TA-Lib)

## Installation

### macOS and Linux

```bash
$ git clone https://github.com/CryptoPirates/ta-lib
$ cd ta-lib
$ ./configure --prefix=/usr
$ make
$ make install
```

#### Ubuntu

```bash
$ git clone https://github.com/CryptoPirates/ta-lib
$ cd ta-lib
$ ./configure --prefix=/usr LDFLAGS="-lm"
$ make
$ make install
```
